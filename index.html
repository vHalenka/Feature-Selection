Experiment Results

This website showcases how different settings affect the performance of a Tsetlin machine learning model on a custom dataset. The dataset has built-in noise that changes based on position, making it more challenging to learn. We tested how changing parameters like noise type, reliability, and model settings impacts the results. You can explore animations showing how the model evolves over time and see noise distributions for each setup.
